# 가마고치 (Gamagotchi)

> 가민 × 다마고치 — 달리면 캐릭터를 살린다

Garmin 스마트워치용 Connect IQ 장치 앱.  
달리기로 토큰을 모아 캐릭터에게 밥을 주고, 시간이 지나면 캐릭터가 성장한다.  
밥을 안 주면 아프고, 그래도 방치하면 사망한다.

---

## 컨셉

- 달리기 → 토큰 획득 → 밥주기로 캐릭터 생존 유지
- 성장은 시간 기반 자동 진화 (알 → 아기 → 어른)
- 48시간 미급식 시 아픈 상태, 추가 24시간 방치 시 사망(리셋)
- 픽셀 아트 스타일 · 각 단계별 정상/아픈 두 가지 상태
- 손목을 들었을 때 캐릭터 애니메이션 재생
- 대상 기기: Garmin Forerunner 255

---

## 캐릭터 — 펭귄

원래 못 달리는 펭귄이 달리기로 성장한다는 서사. 귀여움 최우선.

### 성장 단계

| 단계 | 조건 | 모습 |
|------|------|------|
| 🥚 알 | 시작 | 흰색 동글동글한 펭귄 알 |
| 🐧 아기 | 시간 경과 (자동) | 회색 솜털, 동그란 눈 아기 펭귄 |
| 🐧 어른 | 시간 경과 (자동) | 블랙&화이트, 자신감 넘치는 어른 펭귄 |

### 캐릭터 상태

| 상태 | 조건 |
|------|------|
| 정상 | 48시간 내 밥주기 완료 |
| 🤒 아픈 상태 | 48시간 미급식 |
| 💀 사망(리셋) | 아픈 상태 + 추가 24시간 방치 |

## 기술 스택

- **언어**: Monkey C
- **개발 환경**: VS Code + Connect IQ SDK
- **캐릭터 리소스**: 픽셀 아트 PNG (직접 제작, Aseprite 32×32px)
- **배포**: Connect IQ 스토어
- **API 문서**: https://developer.garmin.com/connect-iq/api-docs/

### 주요 API

| 모듈 | 용도 |
|------|------|
| `Toybox.Background` | 런 종료 감지, 백그라운드 이벤트 |
| `Toybox.Activity` | 런 중 거리 읽기 |
| `Toybox.Application.Storage` | 토큰·상태 영구 저장 |
| `Toybox.Notifications` | 배고픔 알림 |
| `Toybox.WatchUi` | 화면 렌더링, 버튼 입력(밥주기) |
