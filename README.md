# 🥚 가마고치 (Gamagotchi)

> 가민 × 다마고치 — 달리면 캐릭터가 큰다

Garmin 스마트워치용 Connect IQ 워치페이스 앱.  
달리기 누적 거리(km)가 쌓일수록 화면 속 픽셀 캐릭터가 성장한다.

---

## 컨셉

- 달리면 캐릭터가 자동으로 성장 (별도 조작 없음)
- 픽셀 아트 스타일 · 3단계 성장 (알 → 아기 → 어른)
- 손목을 들었을 때 캐릭터 애니메이션 재생
- 대상 기기: Garmin Forerunner 255

---

## 성장 단계

| 단계 | 조건 | 캐릭터 |
|------|------|--------|
| 🥚 알 | 0 ~ 10 km | 노란 알 |
| 🐣 아기 | 10 ~ 50 km | 눈 달린 초록 생명체 |
| 🐸 어른 | 50 km 이상 | 모자 쓴 보라 캐릭터 |

---

## 기술 스택

- **언어**: Monkey C
- **개발 환경**: VS Code + Connect IQ SDK
- **캐릭터 리소스**: 픽셀 아트 PNG
- **배포**: Connect IQ 스토어

---

## 로드맵

- **v1**: 성장 3단계 + 워치페이스 기본 표시
- **v2**: 밥주기 버튼 인터랙션, 멀티 기기 지원
